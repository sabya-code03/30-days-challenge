### ⚡ The Core Approach

Competitive programming aur core CSE development mein speed aur optimization zaruri hai. Is challenge ka main motive sirf code likhna nahi, balki code ko **optimize** karna tha. Har solution direct, logical, aur minimal time/space complexity ko dhyan mein rakh kar likha gaya hai.

### 🛠️ Execution Guide

Agar aap mere daily challenges ko apne system par compile aur run karna chahte ho:

```bash
# Clone the repository
git clone [https://github.com/sabya-code03/30-days-challenge.git](https://github.com/sabya-code03/30-days-challenge.git)

# Jump into the workspace
cd 30-days-challenge

# Compile a specific day's code (e.g., Day 30) using g++
g++ day30.cpp -o day30

# Run the compiled executable
./day30
